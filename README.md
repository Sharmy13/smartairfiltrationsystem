# smartairfiltrationsystem
This project presents a Smart Air Filtration System designed to monitor and improve indoor air quality using ArduinoUno.The system continuously detects air quality levels using an air quality sensorWhen the detected pollution level exceeds a predefined threshold the system automatically activates filtration mechanism through a relay-controlled fan.
The goal of this project is to develop a low-cost and efficient air monitoring and filtration system that can help maintain a healthier indoor environment.

Features:

Real-time air quality monitoring

Automatic activation of filtration system when pollution level increases

Energy-efficient operation (system turns off when air quality improves)

Simple and cost-effective implementation using Arduino

Components Used:

Arduino Uno

Air Quality Sensor

Relay Module

DC Fan / Air Filter

Jumper Wires

Power Supply

Working Principle:
The air quality sensor continuously measures the level of pollutants in the air. The sensor data is sent to the Arduino Uno for processing. If the pollutant level exceeds the set threshold, the Arduino activates a relay module, which turns on the fan or air filter to purify the air. Once the air quality returns to a safe level, the system automatically switches off the filtration unit.

Applications:

Smart homes

Indoor air quality monitoring systems

Offices and classrooms

Low-cost environmental monitoring projects

Future Improvements:
